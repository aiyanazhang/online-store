# 咖啡杯

简单介绍：一只用于盛放热饮（如咖啡、茶）的杯子，常见于家庭、办公室和咖啡馆。设计兼顾保温、手感与美观。

主要特征
- 容量：约 200–400 毫升（常见规格）。
- 材质：瓷、陶、玻璃、不锈钢或搪瓷等。
- 形状：底部较窄、杯壁向上展开，带把手便于握持。
- 功能：保温、防烫、便于清洗，有些带杯盖便于携带。

使用场景
- 日常早餐或工作间隙饮用热饮。
- 咖啡馆为顾客提供热饮容器。
- 办公桌上常备，适合随手饮用。

保养建议
- 陶瓷/瓷器：可用中性洗洁剂手洗或放入洗碗机（留意装饰涂层说明）。
- 玻璃：避免剧烈温差以防裂开。
- 不锈钢保温杯：定期拆开密封圈清洗，避免长时间存放含糖饮料以免异味。
- 若有印花或金属装饰，尽量避免高温或强酸/强碱清洗。

简短示例
- 早晨一杯现磨咖啡 -> 提神、开始一天。
- 办公室加班时的热茶 -> 缓解疲劳、保持清醒。
